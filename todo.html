Stylish To-Do List Manager

Personal Project | Role: Full-Stack Developer
Technologies: HTML5, CSS3, Vanilla JS, LocalStorage, canvas-confetti

This responsive web app lets users manage tasks with flair — persistence, drag-and-drop reordering, due dates, theme toggle, and even celebratory confetti. Designed to feel both functional and fun.

Key Features

  • Task Management: Add, edit, complete, and delete tasks with persistence in localStorage.
  • Due Dates: Optional deadlines displayed beside each entry.
  • Confetti Celebration: A burst of colors when marking tasks completed.
  • Theme Toggle: Switch between light/dark pastel modes.
  • Drag-and-Drop: Intuitive reordering of tasks without libraries.

Skills Applied

  • Responsive frontend development
  • JavaScript DOM manipulation
  • Data persistence with localStorage
  • UI/UX polish with animations
  • Lightweight interactive features
