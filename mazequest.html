Treasure Hunt Maze

University Project | Role: Game Developer

Technologies: C, OpenGL, GLSL Shaders, Texture Mapping, Matrix-Based Collision

Treasure Hunt Maze is a 2D puzzle-adventure game created in C using OpenGL. The player explores a maze filled with treasures, doors, and water obstacles while avoiding collisions with the environment. The game uses a matrix-based collision detection system to determine valid movements and interactions with walls and terrain elements. Custom shaders and texture rendering were used to create the game's colorful, retro-inspired aesthetic.

My Contributions

  • Game Logic & Physics: Implemented player movement, jumping mechanics, and interactions (doors, keys, treasures).
  • Collision System: Designed and integrated a matrix-based collision model for walls and water boundaries.
  • Graphics Pipeline: Set up OpenGL shaders (vertex and fragment) for textured rendering and lighting effects.
  • Input Handling: Created custom keyboard controls for movement, jumping, and interactions.
  • UI & Instructions: Developed the game's start screen, controls display, and user feedback elements.

Skills Applied

  • Low-level programming in C
  • 2D graphics rendering with OpenGL
  • Matrix-based environment and collision logic
  • Shader programming (GLSL)
  • Game loop architecture and event handling
  • Texture mapping and asset management
