Real-Time Object Tracker

University Project | Role: Hardware Designer
Technologies: Arduino UNO, C/C++, HC-SR04 Ultrasonic Sensor, Dual Servos, Breadboard Circuitry

The Object Tracker is a playful, interactive prototype that uses an ultrasonic sensor and two servo motors to follow nearby objects. Each servo is styled like a cartoonish eyeball, giving the system a lifelike, expressive feel. It simulates computer vision behavior using distance sensing and motion logic—no camera needed!

My Contributions

  • Hardware Architecture: Designed and wired the full circuit, including optimal servo layout for realistic eye movement.
  • Servo Control: Calibrated dual-axis motion for smooth, reactive tracking using precise PWM signals.
  • Sensor Integration: Connected and configured the HC-SR04 sensor to detect distance and drive servo logic in real-time.
  • Collaboration: Teamed up with a partner who handled software tuning and logic; ensured seamless hardware/software communication.

  • Dual-axis servo movement control
