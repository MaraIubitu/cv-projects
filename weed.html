Agricultural Weed Detection Robot

Smart Autonomous Weeding Prototype

← Back to Portfolio

University Project | Role: Hardware Designer
Technologies: Raspberry Pi Pico W, MicroPython, L298N Motor Driver, HC-SR04 Ultrasonic Sensor, Dual Servos, LED "Weed Zapper"

The Agricultural Weed Detection Robot is a smart prototype designed to simulate autonomous weeding in a crop field. Equipped with a camera and server-based AI analysis, the robot moves in small steps, scanning for plants. If a weed is detected, it activates a playful "zapper" mechanism using servos and an LED to mimic targeted elimination. The system also integrates an ultrasonic sensor for obstacle avoidance, allowing for smooth autonomous navigation.

My Contributions

  • Hardware Architecture: Designed and wired the full circuit, including 4-motor drive system, ultrasonic sensor, dual servos, and LED zapper.
  • Motor & Power Control: Integrated L298N motor driver with Raspberry Pi Pico W, ensuring stable control from a 7.4V battery source.
  • Sensor Integration: Connected and calibrated the HC-SR04 ultrasonic sensor for real-time obstacle detection and avoidance logic.
  • Servo Mechanism: Configured servo motors for horizontal sweep and vertical tilt to accurately "zap" weeds when detected.
  • Collaboration: Partnered with a teammate who implemented server communication and AI analysis logic; ensured seamless hardware/software integration.
