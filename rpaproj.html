UiPath Hotel Hunter

Personal Project | Role: RPA Developer

Technologies: UiPath Studio, Excel, Web Scraping, Data Processing

The Hotel Hunter is an RPA automation designed to search hotel booking platforms, extract relevant hotel information (name, price, rating, location), and compile the results into structured Excel sheets. It streamlines repetitive search tasks and ensures accurate data collection for travel planning or analysis.

My Contributions

  • Workflow Design: Built UiPath sequences to navigate websites, handle search inputs, and extract hotel data.
  • Data Extraction: Implemented reliable scraping logic using selectors and dynamic data handling.
  • Error Handling: Integrated Try-Catch blocks and logging to handle navigation errors and unexpected popups.
  • Data Management: Automated Excel reporting, ensuring structured storage of hotel details for analysis.

  • Web scraping and data extraction
  • Excel automation and structured reporting
  • Error handling and workflow robustness
